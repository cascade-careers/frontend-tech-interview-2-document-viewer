ADR-007: Authentication Architecture. Status: Accepted. Context: We need a unified authentication system that supports SSO, MFA, and session management across all Cascade applications. Decision: Adopt Firebase Authentication with custom claims for role-based access control. Consequences: All services must validate Firebase ID tokens. Custom claims are set via Cloud Functions on user creation and role changes. Session cookies are used for server-rendered pages. This approach simplifies our auth surface area while maintaining flexibility for future identity provider integrations.
